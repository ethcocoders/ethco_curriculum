# Ethco Academy Curriculum

Welcome to the Ethco Academy Curriculum! This curriculum is designed to provide a comprehensive learning path for aspiring full-stack web developers. It is structured into four main phases, progressing from foundational concepts to advanced topics and career preparation.

## Curriculum Structure

### Phase 1: Foundations

This phase covers the fundamental concepts essential for every web developer.

*   **01 Introduction to Web Development:** Understanding the web, its technologies, and development tools.
*   **02 Git and Version Control:** Mastering Git for collaborative development and code management.
*   **03 HTML and CSS:** Building and styling web pages with the core languages of the web.
*   **04 JavaScript Basics:** Learning the fundamentals of JavaScript programming.

### Phase 2: Front-End Development

This phase focuses on building modern, interactive, and responsive user interfaces.

*   **01 Intermediate HTML & CSS:** Advanced styling techniques, responsive design, accessibility, and animations.
*   **02 JavaScript Deep Dive:** Exploring advanced JavaScript concepts, asynchronous programming, and working with APIs.
*   **03 React Framework:** Developing dynamic single-page applications using the React library.

### Phase 3: Back-End Development

This phase offers two distinct paths for server-side development, allowing students to specialize in their preferred technology stack.

#### Path A: JavaScript (Node.js)

*   **01 Introduction to Node.js:** Building server-side applications with Node.js and the Express framework.
*   **02 Databases:** Working with various database systems (SQL and NoSQL) for data storage and retrieval.
*   **03 Authentication:** Implementing secure user authentication and authorization mechanisms.
*   **04 APIs:** Designing and building RESTful APIs for web services.

#### Path B: Ruby (Ruby on Rails)

*   **01 Ruby Programming:** Learning the elegant and powerful Ruby programming language.
*   **02 Introduction to Ruby on Rails:** Developing robust web applications with the Ruby on Rails framework.
*   **03 Databases with Active Record:** Managing database interactions efficiently using Active Record.
*   **04 Authentication:** Implementing secure user authentication in Rails applications.
*   **05 APIs:** Creating and consuming APIs within the Rails ecosystem.

### Phase 4: Advanced Topics & Career Preparation

The final phase covers advanced development practices and essential skills for launching a successful career in tech.

*   **01 Testing:** Writing effective tests for front-end and back-end applications to ensure code quality.
*   **02 Computer Science:** Understanding fundamental computer science concepts, algorithms, and data structures.
*   **03 Job Search Preparation:** Strategies for resume building, portfolio development, and networking.
*   **04 Interview Practice:** Preparing for technical interviews and coding challenges.
